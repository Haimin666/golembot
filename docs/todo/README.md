# Golem TODO Tracking

This directory tracks all known to-do items, technical debt, and future feature plans.

## File Index

| File | Content |
|------|---------|
| [bugs-and-debt.md](bugs-and-debt.md) | Known Bugs and Technical Debt |
| [phase3-features.md](phase3-features.md) | Phase 3 Feature Plans (Multi-Engine, Skill Ecosystem, etc.) |
| [enhancements.md](enhancements.md) | Enhancements to Existing Features |

## Conventions

- Each TODO is tagged with a priority: `P0` (blocking), `P1` (important), `P2` (quality-of-life), `P3` (nice-to-have)
- Once completed, add `~~strikethrough~~` to the item or move it to the "Completed" section to preserve the record
- When adding a new TODO, include the discovery context and relevant file paths
